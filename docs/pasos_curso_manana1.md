# Pasos del curso — Mañana 1: Aprender a trabajar con Claude Code

Guía operativa para participantes, basada en
`docs/programa_curso_vJulio2026.pdf`. Cada módulo lista qué hacer
tú (el participante) y qué deberías tener listo al terminar.

Ver también [pasos_curso_manana2.md](pasos_curso_manana2.md).

---

> Esqueleto. Se detalla más adelante.

### Módulo 1 · ¿Qué es un agente de IA? (09:00–09:30)
- [ ] Ver demo en vivo y presentación del dataset de Atacama.
- [ ] Recorrer la estructura del repositorio del curso.

### Módulo 2 · Instalación y entorno (09:30–10:30)
- [ ] Instalar y autenticar Claude Code con cuenta Pro.
- [ ] Clonar el repositorio y crear tu branch personal.
- [ ] Checkpoint: Claude Code responde en tu terminal.

### Coffee break (10:30–11:00)

### Módulo 3 · Modelos, tokens y gestión de contexto (11:00–11:30)
- [ ] Practicar `/model` y gestión de sesiones/compactado.

### Módulo 4 · Manejo de archivos y entorno de trabajo (11:30–12:00)
- [ ] Practicar `/add` con el dataset de Atacama.
- [ ] Leer/explorar archivos CSV y TSV desde el agente.

### Módulo 5 · Skills, CLAUDE.md y conectores MCP (12:00–12:30)
- [ ] Comparar al agente con y sin `CLAUDE.md`.
- [ ] Demo conceptual de conectores MCP.

### Módulo 6 · Buenas prácticas de prompting (12:30–13:00)
- [ ] Pedir plan antes de ejecutar, dar contexto científico, pedir
      autoverificación.
- [ ] Cierre del día: `CLAUDE.md` personal finalizado, branch listo.

**Al cierre del día 1:** entorno instalado, branch personal en
GitHub, `CLAUDE.md` con contexto del dataset de Atacama.
